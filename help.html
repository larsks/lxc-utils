lxc-archlinux

This template installs a minimal Arch Linux container using the "pacstrap" tool.

Usage

lxc-archlinux -n|--name=*name* [-p|--path=*path*] [-h|--help] ...
lxc-archlinux -h|--help

Options

Mandatory arguments:

  • -n, --name name -- container name
  • -p, --path path -- path to container directory

Optional arguments:

  • -P, --packages packages -- comma separated list of additional packages to install.
